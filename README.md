Competitive Congressional District Map

A React component with interactive congressional districts built on top of Google Maps Javascript API using FullstackReact's "google-react-map" package. The Google Fusion Table layer provided by Sharon Machlis, using Cartographic Boundary KML Files provided by the United States Census Bureau on the 115th Congress district boundaries.

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).
